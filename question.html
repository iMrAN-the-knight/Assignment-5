Question-1: What are the different ways to select an element in the DOM?

There are 5 different ways to select an element in Dom . 1. document.getElementsByTagName() 2 document.getElementsByClassName() 3 document.getElementById() 4 document.querySelector() 5 document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml returns all texts with html tags that is inside of element

innerText gives back all texts that inside an element and its childs also

textContent has all text inside element and its childs, including hidden text, but it does not interpret HTML tags

Question-3: What is event delegation in the DOM?

Event delegation means adding an event to a parent so it works for its child elements too. This helps when there are so many items or new one is added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event begins at the click of an element, it moves from that element to its parent, and then to higher elements

Question-5: How do you create, add, and remove elements using JavaScript?

Process is Create: let newElement = document.createElement('div'), Add: parent.appendChild(newElement), Remove: newElement.remove().
